# 献立（オフライン版 / kondate offline）

スマホのブラウザだけで動く週次献立アプリ（PC不要）。
献立の計算はブラウザ内の Python（[Pyodide](https://pyodide.org/)）で行います＝PC版と同じエンジン。

## できること
- 来週の夕食／弁当が要る日を選ぶ
- 残り食材を入れて優先的に使い切る献立を生成
- 日ごとに「ジャンル（和洋中エスニック）／肉の種類／別の料理に」変更（その日だけ変わる）
- 買い物リスト（店の順・常備品は除外・チェック保存）
- カレンダーに記録、後からレシピ（材料・分量・手順）を閲覧

## 使い方
1. このページを開く
2. ブラウザの「ホーム画面に追加」でアプリのように使える
3. 初回ロード後はオフラインでも動作（端末内で計算）

## 注意
- **個人利用目的の試作**です。レシピは公開情報を参考にローカル LLM で書き直した私的データで、第三者の利用・再配布は想定していません。
- 実行時に外部送信・LLM 呼び出しはありません（計算はすべて端末内）。
